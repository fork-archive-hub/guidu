## Contributing

Clone the repository

```
git clone https://github.com/uidu-org/guidu
cd guidu
```

Start all packages (not reccommended)

```
yarn start
```

Start specific folders

```
yarn start:forms
yarn start:data
yarn start:dashboards
```

or specific packages

```
yarn start modal-dialog field-geosuggest
```
