## Contributing

Clone the repository

```
git clone https://github.com/uidu-org/guidu
cd guidu
```

Install bolt

```
yarn global add bolt
bolt install
```

Start all packages

```
bolt start
```

Start specific folders

```
bolt start:forms
bolt start:data
bolt start:dashboards
```

or specific packages

```
bolt start modal-dialog field-geosuggest
```
