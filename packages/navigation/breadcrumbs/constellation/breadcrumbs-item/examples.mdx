## Truncation width

When a `truncationWidth` is specified,
long item names will truncate and a [tooltip](/components/tooltip) containing the full item name will appear on hover.
If unspecified,
truncation will only occur when an item cannot fit alone on a line.

<Example
  Component={BreadcrumbsItemTruncation}
  packageName="@uidu/breadcrumbs"
/>
